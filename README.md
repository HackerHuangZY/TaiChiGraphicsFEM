# TaiChiGraphicsFEM
## 仓库介绍
本仓库为太极图形的fem实例
## 环境配置
运行环境为 Python3.9 需要pip taichi库
```bash
pip install taichi
```
## 运行方式
在命令行中 python fem-explicit.py即可运行
```bash
python fem-explicit.py
```
